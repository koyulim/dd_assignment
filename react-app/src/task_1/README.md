# 미션
비동기 작업을 처리하는 방법인 async와 await를 사용하여 API 호출 및 비동기 처리를 구현합니다.  
예를 들어, 영화 정보를 가져오는 API를 사용하여 데이터를 비동기적으로 불러오고 화면에 표시합니다.  
기능이 작동하는 앱 코드와 실행 화면 스크린샷을 결과물로 제출합니다.
