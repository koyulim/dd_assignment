# 미션
React Context를 사용하여 전역 상태를 관리하고, 여러 컴포넌트에서 동일한 데이터를 공유할 수 있도록 구현합니다.  
예를 들어, 사용자 로그인 상태나 언어 설정 등을 관리합니다.  
React Context를 이용한 상태 관리 코드와 실행 화면 스크린샷을 결과물로 제출합니다.
