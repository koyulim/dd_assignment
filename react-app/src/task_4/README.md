# 미션
useEffect를 활용하여 컴포넌트가 마운트 되었을 때와 언마운트될 때의 동작을 구현합니다. 예를 들어, API 데이터를 불러와서 컴포넌트에 표시하는 작업을 수행합니다.  
useEffect를 활용한 데이터 fetching 코드를 결과물로 제출합니다.
