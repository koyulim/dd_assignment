# 미션
Next.js를 사용하여 영화 목록 앱을 여러 페이지로 나누고, 페이지 간 이동이 가능하도록 라우팅을 설정합니다.  
페이지 간 이동이 가능한 \앱 코드와 실행 결과 스크린샷을 결과물로 제출합니다.
